# Contributing to Decision Journal Template

Thank you for helping improve decision journaling for everyone!

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a Pull Request

## Ideas for Contributions

- New template formats (Obsidian, Roam Research, etc.)
- Localized templates in other languages
- Integration scripts for popular note-taking apps
- Decision review automation tools
- Calibration tracking visualizations

## Resources

This project draws inspiration from the decision-making principles at [KeepRule](https://keeprule.com), where timeless wisdom from master investors and thinkers is curated for practical application.
